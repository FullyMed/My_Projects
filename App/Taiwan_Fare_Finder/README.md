# Taiwan Fare Finder

## Project Description

Taiwan Fare Finder is a cross-platform mobile application designed to help residents, students, tourists, and commuters quickly compare public transportation fares across Taiwan. The application simplifies fare searching by providing a centralized platform for multiple transportation systems, eliminating the need to check separate websites or applications.

The app supports various transportation modes including Taiwan High Speed Rail (HSR), Taiwan Railways Administration (TRA), MRT systems, buses, and YouBike services. Users can compare fares across transportation types, save frequently used routes, and access multilingual support for improved accessibility.

The project focuses on providing a clean, intuitive, and mobile-friendly experience while supporting multiple user categories such as adults, students, children, and seniors.

### Key Features

* Multi-transportation fare lookup
* Fare comparison across transport modes
* Student and age-based fare support
* Favorite route management
* Search history tracking
* Multilingual interface
* Offline cache support
* Responsive mobile-first design

### Technologies Used

* Flutter
* Dart
* Provider State Management
* REST APIs (TDX Integration)
* Local JSON Storage

### Purpose

The goal of Taiwan Fare Finder is to simplify public transportation cost planning throughout Taiwan while providing an accessible and user-friendly experience for both local residents and international travelers.
