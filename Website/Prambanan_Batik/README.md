# Prambanan Batik

## Project Description

Prambanan Batik is a full-stack e-commerce website developed to showcase and manage a large-scale batik product catalog. The platform enables customers to browse products, manage shopping carts, place orders, and connect directly to purchasing channels while providing administrators with inventory and product management capabilities.

The system is designed to support a growing product inventory and is optimized for deployment on shared hosting environments using PHP and MySQL. Special attention is given to performance, maintainability, and scalability for handling large catalogs containing hundreds or thousands of products.

The project combines traditional web technologies with database-driven functionality to create a practical online storefront suitable for small and medium-sized businesses.

### Key Features

* Product catalog management
* Category and product search
* Shopping cart functionality
* Order submission system
* Email notification support
* Admin dashboard
* Product inventory management
* SEO-friendly structure
* Shared hosting compatibility

### Technologies Used

* HTML5
* CSS3
* JavaScript
* PHP
* MySQL
* phpMyAdmin

### Purpose

The purpose of Prambanan Batik is to provide a scalable and professional online sales platform that helps batik businesses manage products, improve customer reach, and streamline ordering processes through a modern web-based solution.
