# BoardGames Analyzer

## Project Description

BoardGames Analyzer is a hybrid board game recommendation and analytics platform designed to help users discover board games through explainable recommendation techniques. Unlike traditional recommendation systems that focus solely on prediction accuracy, this project emphasizes transparency by providing meaningful explanations behind every recommendation.

The system combines multiple recommendation approaches, including title-based similarity, trait-based filtering, sentiment analysis, and hybrid recommendation strategies. Users can discover games based on previously enjoyed titles, preferred mechanics, themes, complexity levels, and gameplay characteristics.

The project also includes a complete evaluation framework using ranking metrics such as Recall@K and NDCG@K to measure recommendation effectiveness. A Streamlit web application provides an interactive interface for exploring board game datasets, generating recommendations, and understanding recommendation logic.

### Key Features

* Title-based recommendation engine
* Trait-based recommendation engine
* Hybrid recommendation system
* Explainable recommendation results
* Sentiment-enhanced ranking
* Board game analytics dashboard
* Evaluation using Recall@K and NDCG@K
* Interactive Streamlit interface

### Technologies Used

* Python
* Pandas
* NumPy

* Streamlit

### Purpose

This project was developed as a Data Science and Recommender Systems research project to demonstrate practical machine learning applications, recommendation techniques, and explainable AI concepts within the board gaming domain.
